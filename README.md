**# Proyecto: Sistema de Análisis de Soporte Técnico (SQL Server)

## Descripción
Este proyecto simula un entorno de mesa de ayuda (Service Desk) donde se gestionan incidentes y solicitudes. El objetivo es demostrar la capacidad de modelar datos relacionales y extraer métricas clave para la toma de decisiones operativas.

## Objetivos del Proyecto
* Diseñar un esquema de base de datos relacional para el seguimiento de tickets.
* Implementar consultas para el monitoreo y priorización de incidentes activos.
* Realizar análisis de carga operativa mediante funciones de agregación.

## Funcionalidades principales
- **Modelado:** Creación de tablas normalizadas (`Usuarios`, `Incidentes`).
- **Análisis:** Filtros condicionales para gestión de estados y priorización.
- **Reportes:** Uso de `JOINs` para integrar información de usuarios con sus respectivos reportes.

## Tecnologías utilizadas
- SQL Server (T-SQL)
- SQL Server Management Studio (SSMS)**
